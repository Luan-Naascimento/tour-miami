# Tour Miami 🌴

Projeto desenvolvido para uma apresentação acadêmica na disciplina de **ASLAN (Inglês / ESL)**, com o objetivo de apresentar informações turísticas sobre a cidade de **Miami** utilizando uma aplicação web.

## 📌 Objetivo

O objetivo do projeto foi criar uma interface web simples para auxiliar na apresentação sobre Miami, destacando pontos turísticos, cultura e características da cidade, ao mesmo tempo em que foram praticados conceitos de desenvolvimento front-end.

## 🚀 Tecnologias utilizadas

- JavaScript
- HTML
- CSS

## ⚙️ Funcionalidades

- Página informativa sobre a cidade de Miami
- Estrutura organizada utilizando componentes React
- Interface visual para apresentação de conteúdo turístico
- Layout simples e responsivo
